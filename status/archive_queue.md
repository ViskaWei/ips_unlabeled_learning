## 使用说明

### 查看队列
```
a        # 或 archive
s        # 仅查看队列
```

### 归档操作
```
a 1      # 归档第 1 个
a all    # 全部归档
```

### 归档流程

1. **检查队列**: 查看待归档文件
2. **撰写报告**: 按 `exp.md` 模板结构化
3. **更新文档**: 同步到 roadmap.md 和 hub.md
4. **移动文件**: 原始文件移到 processed/
